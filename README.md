# Claude Courses — Colab Notebooks

Google Colab notebooks adapted from the [Anthropic Academy](https://anthropic.skilljar.com/) courses on Skilljar.

All credit for the original course material goes to Anthropic. These notebooks are a Colab-friendly rehosting for easy classroom use.

## How to use

1. Click any `.ipynb` file in this repo to open it on GitHub.
2. At the top of the rendered notebook, click the **"Open in Colab"** button.
3. The notebook opens in Google Colab — run cells with `Shift+Enter`.
4. To keep your edits, use **File → Save a copy in Drive** in Colab. Your copy is personal and won't sync back here.
5. To pull updates later, re-open the GitHub link (your saved Drive copy won't auto-update).
